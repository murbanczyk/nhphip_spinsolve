Pulse sequence and Arduino code for performing ZQ-COSY and 1D-SEPP experiments with nh-PHIP hyperpolarization on Magritek Spinsolve spectrometer

Folder ArduinoSourceCode contains wiring scheme and code for Arduino board.

Folder PulseSequences contains the pulse sequences SEPP and ZQCOSY compatible with bubling unit and Magritek Spinsolve Expert

Folder Python contains scripts to communicate between the pulse-sequences and Arduino.
